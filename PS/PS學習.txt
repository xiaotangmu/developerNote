1.2 右邊框上面的基本功能 復位

2. 设计师常用格式
jpg（压缩）
png（透明）
gif （动态）
psd (分层，源文件)
2.1 打开文件方法
1. 拖动图片到任务栏图标，再将图片拖动到菜单栏上（工作区以外）
2. ctrl + o

3. 新建文档注意事项
1. ctrl + n
2. 文件 -> 新建
注意事项：
1. 命名要规范
2. 文档类型一般选择默认大小（其他根据具体）
检查设计的单位，印刷用毫米，网页与UI用像素
3. 宽高根据实际
4. 分辨率为72像素/英寸，印刷设计用300
5. 颜色RGB，印刷类用CMYK

4. 文件存储和存储为
存储默认格式为psd, Ctrl +S
存储为：Shift + Ctrl +s
一般备份多个文件

5. 图像的放大与缩小
1. 选择缩放工具，按鼠标左键不放左右摆动鼠标即可放大与缩小
2. Ctrl + +(=) 放大 ，Ctrl + - 缩小（此时不用选中放大镜）
3. 按 Ctrl + 空格键，此时鼠标会变成放大镜，按鼠标左键不放，
放大：向右摆动；缩小向左摆动

6. 画布的移动
放大后才可以拖动
1. 空格按住鼠标拖动
2. 直接使用托手工具

7.移动工具
1. v 快捷使用
对着相应图层可以直接拖动
建立图层组可以拖动整组
2. 自动选择图层或组（工具属性栏）
当文件中有多个图层或组，我们可以在移动工具的情况下，将自动选择
“图层或组”勾上，软件会自动找到对应的组或图层。
如果不勾，可以按住Ctrl 达到自动选择功能
3. 移动工具复制功能
在移动工具的情况下，按alt 键不放，光标变成双箭头，
拖动鼠标移动复制对象即可
4. 跨文件复制
打开多个文件时，使用移动工具选择一个文件，鼠标左键按住不放，托向
文件导航栏中载体对象，进入该文件后，仍不放，拖动到对应位置。
5. 图层对齐功能
居中对齐，对的是中心间距
1. 在移动工具的情况下，按住Shift，可以加选图层（鼠标框中范围），工具属性栏
出现对齐功能，针对选中图层

8. 自由变换功能
快捷键：Ctrl + t
功能：改变所选图层大小、旋转
用法：选择图层，图层边角缩放图层，外面可以旋转

9. 选框工具组及选去的取消
快捷键：m
绘制矩形选区及椭圆选区的工具
绘制正方形及正圆形选区需要按 shift 键
从中心绘制正方形及圆形选区：按 Shift + alt 键
2. 选区的取消
2.1. 绘制完选区后，在选区工具的情况下，点击空白处即可
2.2 绘制完选区后使用快捷键取消：Ctrl + d

3. 选区布尔运算
1. 新选区：如果已经存在选区，那么新绘制选区后会替代原来的选区（默认，特点可以移动选框）
2. 添加到选区：可将新的选区添加到原来的选区中（Shift键）
3. 从选区中减去：可将当前创建的选区从原来的选区中减去（Alt键）
4. 与选区交叉：新建选区时只保留原有选区与创建选区交叉的部分（alt + shift）

10. 填充颜色
在ps中分为前景色和背景色，点击可以更换颜色
填充前景色：Alt + 退格
填充背景色：Ctrl + 退格
填充自定义颜色/内容：shift + 退格
注意：填充颜色前一定要养成新建图层的习惯
没画一个东西建立一个新图层

11. 选框固定大小
在选框工具情况下，在工具属性栏，样式固定大小，设置宽高像素

12. 选区的修改
在选区出现后，菜单栏，选择，修改
1. 边界 （类似html的 border）
出现两个虚线框，此时操作的就是中间范围 -- 边界
2. 平滑
效果：去角，让框更加圆润，类似html 的border-radius
3. 扩展
加大选区, 但周围样式不会改变
4. 收缩
选区，缩小，但周围样式不会改变
5. 羽化
选区变为平滑，填充颜色后，是由中心到四周的渐变色

13. 关于撤销
1. 菜单栏 编辑 - 后退一步
Ctrl + alt + z
2. 打开历史面板，对操作的步骤进行反复撤销操作

14. 图层基础知识
1. 隐藏/显示
点击图层眼睛图标
2. 复制图层
2.1 菜单栏，图层，拷贝图层
2.2 在图层面板，将图层拖到新建图层图标
快捷：ctrl + j
2.3 在图层选择工具情况下，按住 alt 拖动

3. 新建图层
1. Ctrl + shift + n（带对话框）
2. 点击图面面板的新建按钮
3. ctrl + shift + alt + n（快速建立图层，不带对话框）

4. 调整图层顺序
1. 菜单栏 - 图层 - 调整到下一层
2. 在图层面板中，直接拖动改变顺序
3. 向下移一层 Ctrl + [
向上移一层 Ctrl + ]
置到顶层 Ctrl + shift + ]
置到底层 Ctrl + shift +[

5. 图层合并和图层选择
在图层面板中同时选中多个图层（Ctrl/Shift），菜单栏图层
合并图层
Ctrl + e

6. 图层锁定
1. 锁定透明像素：透明区不能编辑
2. 锁定像素：不能进行任何像素的编辑(改变颜色、删除颜色) 可以移动位置
3. 锁定位置：不有移动位置
4. 画板防止嵌套：如果新建的是画板文档，锁定后不能将画板1中元素移动到另一画板
5. 锁定全部：不能对当前图层作任何更改

15. 套索工具组
快捷键：L
1. 套索工具
功能：可以创建不规则选区
用法：直接使用鼠标左键不放到开始位置即可
2. 多边形套索工具
功能：绘制直角或尖角的选区
用法：单击鼠标左键松开，反复同一步操作，直至完成。如果出现误操作时可以按
 退格键/delete 键 来删除点
3. 磁性套索工具
功能：工具可以自动识别背景与前景的边缘部分，使用于背景色单一或者边缘清晰对比较强的图像
用法：单机鼠标左键松开鼠标顺应所选的独享边缘，直至完成

（拖动到其他地方：
套住图以后 选择移动工具 点上图 按住鼠标左键不放 拖到另一张图里；
按Ctrl任何工具状态（除钢笔，路径工具）移动；
保存到新图层：
直接在图层面板，ctrl + j 即可复制选区为新图层；
如果要进行选区交叉，删减，增加，参照选框操作& | ）

16. 透明图与图像合成
1. 透明图
一张图片，将我们需要的地方保留，不需要的制作成透明。
重点：透明图一定要存储为 psd 或者 png 格式才支持透明

2. 图像合成
将两张或多张图，放到一起，形成一个新的画面。
方法：
1. 要打开一个图像或新建一个画布
2. 将我们需要合成的图片直接拖动到新的画面中组成一个新的画面。

17. 快速选择工具（& | 一样）
快捷键：W
1. 快速选择工具
功能：快速选择需要载入选区的部分
用法: 单机鼠标左键拖动，向外拉扩展选区，在使用过程中根据需要不断调整笔触大小，
笔触大小调整方法：“左括号”变小；“右括号”变大，注意调整为英文输入法

2. 魔棒工具
容差：32，
功能：可以快速选择背景单一的色彩区域。
容差：选择的颜色范围
连续：如果选择连续那么不连续的颜色将不会被选中

18. 渐变工具
快捷键：G
阐述：渐变工具应用十分广泛，它可以填充到选区或者整个图层、
蒙版、通道、叠加到图层，也可以填充形状或者描边渐变工具不能
用于索引和位图模式。

点击工具属性栏渐变条弹出预设区
预设区：
1. 软件自带的常用渐变库
设置不透明度
选择需要设置的不透明度色标，在下面不透明度框中输入数据即可，
删除向下拉，添加是点击空白处
设置色标
选择色标，双击或者单击点击颜色更改即可
添加色标
在色标轨道上任意位置点击即可添加色标
删除色标
选择要删除的色标，向下拖动即可删除
色标的位置
选择色标，拖动到合适的位置，也可以输入精确位置数据
复制色标
选中色标，点击空白处，也可以按 alt 键拖动复制

2. 渐变类型
工具属性栏 -- & | 
线性渐变
功能：以知县或角度线的形式渐变效果
径向渐变
以起点到终点为半径，绘制中心向四周扩散或放射的形式表达渐变
角度渐变
以知县为起点，顺时针的形式呈现渐变效果
对称渐变
以中心为轴，向两边延伸出相同渐变
菱形渐变
