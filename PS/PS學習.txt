2. 设计师常用格式
jpg（压缩）
png（透明）
gif （动态）
psd (分层，源文件)
2.1 打开文件方法
1. 拖动图片到任务栏图标，再将图片拖动到菜单栏上（工作区以外）
2. ctrl + o

3. 新建文档注意事项
1. ctrl + n
2. 文件 -> 新建
注意事项：
1. 命名要规范
2. 文档类型一般选择默认大小（其他根据具体）
检查设计的单位，印刷用毫米，网页与UI用像素
3. 宽高根据实际
4. 分辨率为72像素/英寸，印刷设计用300
5. 颜色RGB，印刷类用CMYK

4. 文件存储和存储为
存储默认格式为psd, Ctrl +S
存储为：Shift + Ctrl +s
一般备份多个文件

5. 图像的放大与缩小
1. 选择缩放工具，按鼠标左键不放左右摆动鼠标即可放大与缩小
2. Ctrl + +(=) 放大 ，Ctrl + - 缩小（此时不用选中放大镜）
3. 按 Ctrl + 空格键，此时鼠标会变成放大镜，按鼠标左键不放，
放大：向右摆动；缩小向左摆动

6. 画布的移动
放大后才可以拖动
1. 空格按住鼠标拖动
2. 直接使用托手工具

7.移动工具
1. v 快捷使用
对着相应图层可以直接拖动
建立图层组可以拖动整组
2. 自动选择图层或组（工具属性栏）
当文件中有多个图层或组，我们可以在移动工具的情况下，将自动选择
“图层或组”勾上，软件会自动找到对应的组或图层。
如果不勾，可以按住Ctrl 达到自动选择功能
3. 移动工具复制功能
在移动工具的情况下，按alt 键不放，光标变成双箭头，
拖动鼠标移动复制对象即可
4. 跨文件复制
打开多个文件时，使用移动工具选择一个文件，鼠标左键按住不放，托向
文件导航栏中载体对象，进入该文件后，仍不放，拖动到对应位置。
5. 图层对齐功能
居中对齐，对的是中心间距
1. 在移动工具的情况下，按住Shift，可以加选图层（鼠标框中范围），工具属性栏
出现对齐功能，针对选中图层

8. 自由变换功能
快捷键：Ctrl + t
功能：改变所选图层大小、旋转
用法：选择图层，图层边角缩放图层，外面可以旋转

9. 选框工具组及选去的取消
快捷键：m
绘制矩形选区及椭圆选区的工具
绘制正方形及正圆形选区需要按 shift 键
从中心绘制正方形及圆形选区：按 Shift + alt 键
2. 选区的取消
2.1. 绘制完选区后，在选区工具的情况下，点击空白处即可
2.2 绘制完选区后使用快捷键取消：Ctrl + d

3. 选区布尔运算
1. 新选区：如果已经存在选区，那么新绘制选区后会替代原来的选区（默认，特点可以移动选框）
2. 添加到选区：可将新的选区添加到原来的选区中（Shift键）
3. 从选区中减去：可将当前创建的选区从原来的选区中减去（Alt键）
4. 与选区交叉：新建选区时只保留原有选区与创建选区交叉的部分（alt + shift）

10. 填充颜色
在ps中分为前景色和背景色，点击可以更换颜色
填充前景色：Alt + 退格
填充背景色：Ctrl + 退格
填充自定义颜色/内容：shift + 退格
注意：填充颜色前一定要养成新建图层的习惯
没画一个东西建立一个新图层

11. 选框固定大小
在选框工具情况下，在工具属性栏，样式固定大小，设置宽高像素
