vim基础操作

vim是从 vi 发展出来的一个文本编辑器 。代码补完、编译及错误跳转等方便编程的功能特别丰富
1、进入插入模式命令（在命令模式下操作）:

i: 插入光标前一个字符 

I: 插入行首 

a: 插入光标后一个字符 

A: 插入行未 

o: 向下新开一行,插入行首 

O: 向上新开一行,插入行首

 

2、进入命令模式（在文本模式或者末行模式下）:

ESC:从插入模式或末行模式进入命令模式


3、移动光标（在命令模式下操作）:

h: 左移 

j: 下移 

k: 上移 

l: 右移

M: 光标移动到屏幕中间行 

L: 光标移动到屏幕最后一行行首 

G: 移动到指定行,行号 G（比如移动到22行：22G）

w: 向后一次移动一个字 

b: 向前一次移动一个字

{: 按段移动,上移 

}: 按段移动,下移

Ctrl+d: 向下翻半屏 

Ctrl+u: 向上翻半屏

Ctrl+f: 向下翻一屏 

Ctrl+b: 向上翻一屏

gg: 光标移动文件开头 

G: 光标移动到文件末尾

shift+6  光标到当前行的行首
shift+4  光标到当前行的行末



4、删除命令（在命令模式下操作）:

x: 删除当前光标一个字符,相当于 Del 

X: 删除光标前一个字符,相当于 Backspace

dd: 删除光标所在行,n dd 删除指定的行数（光标所在的行和之后的n行） D: 删除光标后本行所有内容,包含光标所在字符 

d0: 删除光标前本行所有内容,不包含光标所在字符

dw: 删除光标开始位置的字,包含光标所在字符,如果是中文，则会删除光标和光标所在的句子


5、撤销命令（在命令模式下操作）:

u: 一步一步撤销 


Ctrl+r: 反撤销



6、重复命令（在命令模式下操作）:

.: 重复上一次操作的命令



7、文本行移动（在命令模式下操作）:

>>: 文本行右移 

<<: 文本行左移



8、复制粘贴（在命令模式下操作）:

yy: 复制当前行,n yy 复制 n 行【光标之后的n行（含光标所在行）】

p: 在光标所在位置向下新开辟一行,粘贴



9、可视模式（在命令模式下操作，可整块删除、复制、移动）:

v: 按字符移动,选中文本 

V: 按行移动,选中文本可视模式可以配合 d（删除）, y（复制）, >>（右移）, <<（左移） 实现对文本块的删除,复制,左右移动



10、替换操作（在命令模式下操作）:

r: 替换当前字符 

R: 替换当前行光标后的字符


11、查找命令（在末行模式下）:

/: str查找（在shift+:进入末行模式下，然后"/fe",意思就是查找fe字符串）
n: 下一个（这个下查找情况下，很多结果情况下进行往下查找）
N：上一个（这个下查找情况下，很多结果情况下进行往下查找）
:set hls 打开高亮
:set nohls 关闭高亮


12、替换命令（在末行模式下）：

例如：把abc全部替换成123
末行模式下，将所以提议的abc替换成123

:%s/abc/123/g


末行模式下，将第一行至第10行之间的abc替换成123

:1, 10s/abc/123/g


13、vim里执行 shell 下命令（末行模式下）:

末行模式里输入!,后面跟命令
放大缩写

ctrl -  			缩小
ctrl shift +  		放大


ubuntu vim tab 4个空格

编辑文件

sudo vim /etc/vim/vimrc

set ts=4
set expandtab
